## Installation

Add lita-applause to your Lita instance's Gemfile:

``` ruby
gem "lita-applause"
```

## Configuration

There are no required configurations.

## Usage

Applause will trigger whenever the following are said:

```
% applause
% applaud
% slow clap
% bravo
```
